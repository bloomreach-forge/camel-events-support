Introduction

What is Apache Camel - Hippo Events Support?

Apache Camel - Hippo Events Support provides hippoevent: Apache Camel component, Camel Repository Scheduler Job component which can invoke a Camel Endpoint URI, and utility classes to help integration between Hippo CMS/Repository and Apache Camel.

The hippoevent: component is used for subscribing HippoEvent through Hippo Event Bus.

The Camel Repository Scheduler Job component can be configured for a scheduled task and it can invoke a Camel Endpoint URI based on configuration. This makes it easier to create a scheduler task based on Hippo Repository Scheduler.

The demo project contains many useful example scenarios, solutions and other information. Please see About Demo Project for detail.
